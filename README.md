# Mahsa Alert Data

Daily snapshots of Mahsa Alert map layers in GeoJSON format, automatically published from the live database.

## Structure

Each day's data is in a folder named by date:

    2026-03-17/
      incidents.geojson
      protests.geojson

## Source

Data is exported daily from [mahsaalert.app](https://mahsaalert.app) — a platform tracking human rights incidents in Iran.
